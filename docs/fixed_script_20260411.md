# 固定新闻稿正文（唯一事实来源）

## 原稿全文（5 段）

3 月 26 日，济南市人社局主办的"人社服务大篷车"集成宣传活动，在济南美团服务中心山大站开展。

活动以"走进奔跑者——保障与你同行"为主题，聚焦外卖骑手等新就业形态劳动者，把人社服务送到一线。

活动现场，工作人员和志愿者通过发放宣传资料、面对面讲解等方式，向外卖小哥介绍社保参保、权益保障等相关政策内容。

现场还设置了互动环节，让大家在轻松的氛围中了解政策、增强维权意识。

济南市人社局以更加贴近一线、贴近群众的方式，持续推动人社服务走近新就业形态劳动者，打通服务保障"最后一公里"。
