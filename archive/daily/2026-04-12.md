## 二、关键失败路径回顾

### v5 shim 假 ffmpeg 版本（作废）

**问题**:
- `bin/ffmpeg.exe` 和 `bin/ffprobe.exe` 各只有 384 KB
- 这是 Chocolatey shim/包装器，不是真正的 ffmpeg 二进制

**处置**:
- ❌ v5 包作废，不再使用

### v6 真 ffmpeg/ffprobe 带入版本

**进展**:
- 从 gyan.dev 下载官方 ffmpeg 构建
- `bin/ffmpeg.exe` 约 96 MB，`bin/ffprobe.exe` 约 96 MB

**遗留问题**:
- ffprobe JSON 解析逻辑失败
- 报错：`the JSON object must be str, bytes or bytearray, not NoneType`

### v7 修复 ffprobe JSON 解析

**修复内容**:
- 明确使用 `stdout=subprocess.PIPE` 和 `stderr=subprocess.PIPE`
- 添加 stdout 是否为空的检查
- 添加 JSON 解析异常处理 (`json.JSONDecodeError`)
- 添加详细调试输出

**验证结果**:
- 47 条素材中 41 条成功读取
- 成功率 87.2%
